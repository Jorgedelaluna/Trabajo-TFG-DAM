# Proyecto TFG - DAM

## Titulo del Proyecto

CrossFit Manager App

### Descripción de la Idea del Proyecto

Desarrollar una aplicación de gestión para un gimnasio de CrossFit que facilite las operaciones administrativas, mejora la experiencia del usuario y optimice el control de actividades.

### Objetivos

- [ ] Analizar los requisitos funcionales y técnicos necesarios para la gestión de un gimnasio de CrossFit.
- [ ] Diseñar la base de datos y la arquitectura del sistema.
- [ ] Implementar la aplicación utilizando tecnologías modernas (por ejemplo, Kotlin con Android Studio, Spring Boot, Firebase, Worker de Cloudflare o framework web).
- [ ] Realizar pruebas funcionales y de usabilidad.
- [ ] Evaluar la mejora de eficiencia frente al sistema manual actual.
- [ ] Mejorar la experiencia de Usuario e interfaz de la aplicación.
